The Web

* The web is a collection of HTML documents. It was invented in the early 1990's and has an estimated 30 billion pages. (See video)

* The major components of the Web are you, your computer which is running a browser, the Internet, and the servers. The servers communicate using a protocol called HTTP. When a person is accessing a web page, the browser sends requests through the internet to the server which hosts the web page. The server then responds with files which the browser interprets and displays on the screen. (See video)

HTML Markups

*Elements : HTML markups are made up of elements. Elements are composed of opening and closing tags and content. (See video) 

 
     Structure:

     Examples:

      Structure

      Examples

         <img src = "url" alt = "text">: Image tag: used to add pictures See video

Whitespace

Whitespaces are tabs and empty spaces. When HTML processes text on multiple lines, HTML renders it on one line. 

In order to make multiple lines, you use: 

      <br> break tag (See video) 
      <p>  paragraph tag (See video)

Inline and Block

HTML Document Structure 

The structure looks like this: 
    < !DOCTYPE HTML > 
    < html > 
    < head > 
    < title > title < title > 
    < /head > 
    < body > 
    ....content.... 
    < /body > 
    < /html >

The Doctype tag tells what type of HTML that is running the program.
The "HTML" stands for HTML5. The opening and ending HTML tags enclose the entire document.
The opening and ending "head" tags enclose the title of the document.
The opening and ending "body" tags enclose the content of the document from text to hyperlinks to images. This video lays out the concept of document structure.
